# YouTube下载器

基于PySide6和yt-dlp的YouTube视频下载器，提供图形用户界面，支持多种格式选择、字幕下载和封面下载。

## 功能

- 输入YouTube视频链接，获取视频信息
- 选择视频和音频格式
- 选择字幕语言
- 下载视频封面
- 进度实时显示

## 系统要求

- Windows系统
- Python 3.7+
- PySide6
- yt-dlp

## 安装

1. 克隆仓库
```
git clone https://github.com/claudehotline/youtube-downloader.git
cd youtube-downloader
```

2. 安装依赖
```
pip install -r requirements.txt
```

3. 运行程序
```
python app.py
```
